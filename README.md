# Instalacion
 
## Clonar Repositorio

```
$ cd ~
$ git clone https://github.com/Gilgamesh06/Gilgamesh.git
```

## Instalarlo

```
$ cd ~/Gilgamesh
$ chmod 777 Install.sh
$ ./Install.sh
```
